# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0

Initial standalone release of `A-Share Alpha Engine`.

Included in this milestone:

- extracted `qstrategy_v2` into an independent repository
- removed dependency on the legacy `qstrategy` package
- added standalone `Eastmoney` client inside `qstrategy_v2`
- preserved representative baseline backtest, paper account, and daily review outputs
- added project homepage README
- added version roadmap
- added repository workflow and branch conventions
- kept core capabilities:
  - backtesting
  - parameter optimization
  - daily review generation
  - paper account tracking
